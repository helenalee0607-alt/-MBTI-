# MBTI Multi-Agent Interview Simulator （MBTI 多面试官模拟器）

一个基于 **DeepSeek Chat API** 的终端交互式“多面试官模拟面试”小工具。  
你可以选择自己的 **应聘岗位** 和 **公司领域方向**，然后由 5 位不同 MBTI 风格的虚拟面试官（ENTJ / INTP / ENFP / ISTJ / INFJ）轮流对你进行提问、追问、情境题考察，并给出综合评价，帮助你练习面试表达与思考。

---

## ✨ 功能特点

- **多面试官阵容**：  
  - ENTJ：决断、结果导向  
  - INTP：逻辑与推理  
  - ENFP：动机与价值观  
  - ISTJ：细节与执行  
  - INFJ：人际与情绪洞察  

- **公司领域可选**：  
  在开始时输入你想面试的“公司领域方向”，例如：  
  - 互联网金融  
  - 消费电商  
  - 游戏  
  - 智能制造  
  - 教育科技  
  - SaaS 企业服务  
  …

- **自动生成公司背景 & 行业知识**：  
  程序会基于「岗位 + 领域」自动生成一段虚构但合理的公司背景说明，包括：
  - 领域 & 商业模式简介  
  - 公司规模、发展阶段、产品形态  
  - 典型客户 & 业务场景  
  - 团队文化与公司调性  
  - 该岗位在此类公司里的核心工作重点  

- **多轮面试流程设计**：
  1. 面试官风格自我介绍  
  2. **选择题热身**（每位面试官出一题 4 选 1）  
  3. 快速开放式提问  
  4. 基于你某个回答的深度追问  
  5. 情境题 + 其他面试官即时点评  
  6. 每位面试官的正式书面评价  
  7. 自我复盘引导  

- **API Key 安全设计**：  
  不在代码中写死 Key，统一通过环境变量 `DEEPSEEK_API_KEY` 读取，方便上传到 GitHub 而不泄露密钥。
